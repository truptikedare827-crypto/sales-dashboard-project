Project Description

This project presents an Online Shopping Dashboard developed to analyze customer behavior and purchasing patterns in an e-commerce environment. The dashboard provides interactive visualizations to help understand how users interact with an online shopping platform, what factors influence purchases, and how business performance can be improved.

The main objective of this project is to extract meaningful insights from customer session data and support data-driven decision-making for improving conversion rates and user experience.

🛠️ Tools Used
Power BI – for building interactive dashboard and visualizations
Microsoft Excel / CSV – for dataset handling and preprocessing
GitHub – for project hosting and version control
📁 Dataset Information
Dataset Name: Online Shoppers Intention Dataset
File Format: CSV (online_shoppers_intention.csv)
Description:
This dataset contains information about user sessions on an online shopping website. It includes various attributes related to user behavior, session duration, page visits, and whether the session resulted in a purchase.
Key Features:
Administrative, Informational, and Product-related page visits
Bounce rate and exit rate
Page values and session duration
Traffic type and visitor type (new/returning)
Special day indicator
Purchase outcome (target variable)
Purpose of Dataset:
To analyze customer behavior and predict whether a user will make a purchase based on their browsing activity.
